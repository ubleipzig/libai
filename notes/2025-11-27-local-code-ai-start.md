# Starting points for local text review and code assistance

Quick pointers for text review and code assistance.

* [ollama](https://ollama.com), generic CLI tool for interacting with local and cloud-hosted models
* [qwen-code](https://github.com/QwenLM/qwen-code), forked from [gemini-cli](https://github.com/google-gemini/gemini-cli), agentic code cli
* [roo](https://roocode.com/), VS Code plugin, can use various models and providers, support for roles, tools, MCP, etc.
* [jan.ai](https://www.jan.ai/), desktop application, chat, roles, various models and providers
